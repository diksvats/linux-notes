# linux-notes

Self-study notes and hands-on exercises from my Linux/SRE fundamentals learning path, part of a broader 18-month roadmap moving from Application Packaging/VDI into Cloud, SRE, and DevOps roles.

## Setup

- WSL2 + Ubuntu 24.04 LTS
- VS Code (Windows side) for editing
- All exercises run and tested locally before being committed here

## What's in this repo

Each day covers a focused Linux/SRE topic, with:
- Core commands and short explanations
- Hands-on exercises with real terminal output
- A short section on why the topic matters in an SRE/ops context

## Note

These are self-study notes and exercises, not professional/production experience. They reflect ongoing learning as part of my transition toward cloud infrastructure and SRE roles.
